diff --color-words

ワード単位のDIFF

log --graph

ログをツリー表示する

rebase -i <commit>

過去のコミットをいじれる

cherry-pick <commit>

他のブランチの特定のコミットをつまみ食い

blame <path>

ファイルを変更したユーザを特定

gitk

rm <path>

mv <path> <path>

show --stat

pull --rebase

mergeコミットがのこらない

merge --no-ff <commit>

ffせずに必ずマージコミットが残る

reset --hard <commit>

変更がコミットされてないファイルを無視してリセット

clean -fd

コミットされてない奴らをすべて消す

commit --amend

直前のコミットを修正する

add -p

config -l

branch -r
